📱 Patient App
How patients connect to ChemoCure
1. Share the Patient App file
Send the ChemoCure_Patient_App.html file to the patient (via email/WhatsApp).

2. Patient opens it in Chrome
The app works directly in the browser — no installation needed.

3. Install as app (optional)
Chrome → "Add to Home Screen" or "Install App" for a native feel.

4. Connection via same browser
Both files share data when opened in the same browser profile. For cross-device sync, both parties must use the same browser.
💡 Tip: When you add a patient, their MRN and password are auto-generated. Share those credentials with the patient for the app login.
